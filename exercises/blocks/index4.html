Amazing Audi
Detail
The available Audi virtual cockpit is a fully digital 12.3-inch instrument display that allows the driver to customize information such as navigation and audio, offering classic and infotainment modes as well as access Google Earth™ imagery. When it’s go time, the additional sport mode features a large, central tachometer along with performance features like a lap timer and lap-by-lap statistics, inspired by Audi motorsport heritage.
